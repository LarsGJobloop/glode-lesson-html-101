HTML 101

Blokk- og Innline Elementer

Blokk elementer tar hele skjermbredden. Og blir plassert opppå hverandre.

Innline elementer blir plassert etter hverandre. Og nettlesern bestemmer hvordan de skal brytes opp.

Semantisk HTML

HTML elementer har en betydning. Denne betydning blir brukt til å støtte opp under universell utforming, og når vi prøver å forstå nettsider ved hjelp av programmerings språk.

CSS 101

Mens HTML dokumentet beskriver inneholdet på siden gjennom alle de forskjellige elementene. Så beskriver CSS hvordan alt skal se ut. Det er CSS vi bruker for å endre form, farge og alt annet visuelt for elementene.

CSS kommer i hovedsak fra en seperat fil. Vi linker den inn via head seksjonen i HTML dokumentet.

CSS består av to segmenter. Første er at vi må velge hvilket element vi ønsker å endre på (CSS Selector) det andre er en liste ting vi ønsker å endre på og hva vi ønsker å endre de til (CSS Attributes).
